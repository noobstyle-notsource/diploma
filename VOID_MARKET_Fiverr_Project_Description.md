## 🎯 Төслийн Ерөнхий Танилцуулга

**VOID_MARKET Terminal** бол retro-brutalist загварын, дэвшилтэт технологиор хангагдсан тоглочинд зориулсан онлайн маркетплейс интерфэйс юм. Энэхүү систем нь гамерын хэрэгцээнд зориулж gaming gear, boosting сервис, coaching, аккаунт түрээс зэрэг үйлчилгээг нэг дор цуглуулсан платформ болно.

### **Технологийн Стек:**
- **Frontend:** React 19 + TypeScript
- **Styling:** Tailwind CSS + Custom CSS Variables
- **Animation:** Motion/Framer Motion
- **Build Tool:** Vite 6
- **UI Components:** Lucide React Icons
- **Backend Integration:** Express.js + Google Gemini AI

```

├── App.tsx (Main Application)
├── components/
│   └── TerminalPrimitives (UI Components)
├── index.css (Styling)
└── main.tsx (Entry Point)
```

### **Key Components:**
- `Scanlines` - CRT эффект
- `TerminalBorder` - Terminal frame
- `GlitchButton` - Interactive buttons
- `BlinkingCursor` - Terminal cursor
- `ProgressBar` - Loading states
- `RetroSignal` - Signal interference effects
