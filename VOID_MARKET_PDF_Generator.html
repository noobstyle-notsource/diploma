🎯 Төслийн Ерөнхий Танилцуулга

VOID_MARKET Terminal бол retro-brutalist загварын, дэвшилтэт технологиор хангагдсан тоглочинд зориулсан онлайн маркетплейс интерфэйс юм. Энэхүү систем нь гамерын хэрэгцээнд зориулж gaming gear, boosting сервис, coaching, аккаунт түрээс зэрэг үйлчилгээг нэг дор цуглуулсан платформ болно.

Frontend: React 19 + TypeScript
Styling: Tailwind CSS + Custom CSS
Animation: Motion/Framer Motion
Build Tool: Vite 6
UI Components: Lucide React
Backend: Express.js + Gemini AI
